COIL Milestone 1 Submission
Group Members: Darshita, Pratham, Fenish, Megha

This project implements the PktDef class for the robot application layer protocol in C++ using Microsoft Visual Studio.

Contents:
- Main C++ project with PktDef implementation
- MSTest project with unit tests

How to run:
1. Open the solution file in Visual Studio 2022
2. Build the solution
3. Open Test Explorer
4. Run all tests

Notes:
- Milestone 1 covers protocol definition and testing only
- No socket or webserver functionality is included in this submission
